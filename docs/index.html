Inspiration

This project was inspired by many factors; however, my main inspiration was Ketan! I've always wanted to make a project like this that combines programming with something physical (like Arduino and Raspberry Pi), and that combined with the fact that Ketan is the best candidate for this (since I think that his smooth moves are perfect for a bartender (especially a Java's papa C++ one)).

Purpose

This project uses the Serial Monitor to get a specific concentration of drinks from the robotic bartender
